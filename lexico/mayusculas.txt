¿QUÉ ES EL ORDENAMIENTO DE DATOS? ES EL PROCESO DE ARREGLAR DATOS DE CUALQUIER TIPO EN FORMA DESCENDENTE O ASCENDENTE.
¿POR QUÉ DEBERÍAMOS DE ORDENAR? EL ORDENAMIENTO DE DATOS SIRVE PARA OPTIMIZAR LA BÚSQUEDA DE DATOS U ORDENAR RANGOS DE DATOS.
EN C++ EXISTE LA FUNCIÓN PREDEFINIDA SORT() PARA EL ORDENAMIENTO DE DATOS.
BIBLIOTECAS: #INCLUDE <BITS/STDC++.H> Y #INCLUDE <ALGORITHM>
SINTAXIS PARA ORDENAR UN ARREGLO DE FORMA ASCENDENTE: SORT(NOMBREARREGLO, NOMBREARREGLO + N) DONDE N ES EL TAMAÑO DEL ARREGLOS
IMPORTANTE: EL PRIMER PARÁMETRO DE LA FUNCIÓN NOMBREARREGLO Y EL SEGUNDO PARÁMETRO NOMBREARREGLO + N DEFINEN UN RANGO [INICIO, FIN), LO QUE SIGNIFICA QUE EL INICIO DE RANGO ES INCLUSIVO Y EL FIN ES EXCLUSIVO.
